# 💃 AI-Based Dance Analysis System

### 🚀 Smart India Hackathon (SIH) Project

---

## 📌 Project Overview

This project is an **AI-powered Dance Analysis System** designed to analyze classical dance performances (such as Bharatanatyam) using computer vision techniques.

The system processes dance videos and generates **detailed analytical reports** that evaluate:

* Body posture
* Movement patterns
* Visual consistency
* Performance quality

---

## ❗ Problem Statement

Traditional dance evaluation:

* Requires expert judges
* Is subjective and inconsistent
* Lacks automated feedback systems

---

## 💡 Proposed Solution

We developed an AI-based system that:

* Takes a dance video as input
* Processes frames using computer vision
* Extracts meaningful patterns
* Generates structured PDF reports

This helps:
✅ Dancers improve performance
✅ Provide objective evaluation
✅ Support training and learning

---

## 🧠 Key Features

* 🎥 Video-based dance analysis
* 📊 Automated report generation (PDF)
* 🤖 AI-based motion understanding
* 📈 Performance evaluation
* 🧾 Visual analysis reports

---

## 🏗️ System Workflow

1️⃣ Input dance video
2️⃣ Frame extraction
3️⃣ Feature analysis (pose, motion)
4️⃣ Pattern recognition
5️⃣ Report generation (PDF output)

## ⚙️ Installation & Setup

### Step 1: Clone Repository

```bash
git clone https://github.com/your-username/SIH-Dance-Analysis-System.git
cd SIH-Dance-Analysis-System
```

---

### Step 2: Create Virtual Environment

```bash
python -m venv venv
```

Activate:

```bash
venv\Scripts\activate      # Windows
source venv/bin/activate   # Mac/Linux
```

---

### Step 3: Install Dependencies

```bash
pip install -r requirements.txt
```
